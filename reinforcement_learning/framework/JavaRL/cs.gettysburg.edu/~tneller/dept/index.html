Account Configuration

  • The current standard account configuration is in directory ~rtosten/Accounting/user.  Note that most files there are hidden (start with a "."). To see them use "ls -a".  Make sure you have all of these files to keep up-to-date.

  • Setting up JDE:
    1. We highly recommend having your shell changed to bash if your account was created before Spring 2001.  See professors Neller or Presser to have your account shell changed.  If you change your shell and copy the files in the current account configuration to your home directory, you should be all set.
    2. If you choose not to enter the blissful realms of bash, you can follow the directions on Prof. Presser's JDE page.
