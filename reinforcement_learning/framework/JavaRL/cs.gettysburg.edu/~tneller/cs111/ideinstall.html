Eclipse IDE Installation

While not necessary for this course, many students like to install Eclipse on their own machines.  For this, you need to follow the installation instructions for the following two free software packages in this order:

  1. Java Standard Development Kit (a.k.a. JDK, SDK, Java SE Development Kit)
    • Note: Watch out for the bundled bloatware or browser preference changes that Oracle has pulled in the past.
  2. Eclipse
    • Note: Generally, the download is in the form of a .zip file that contains the precompiled applications in the root folder.  Thus installation consists of unzipping the download where you want the application installed.

Even more optional: If you wish to further emulate your experience on our Linux system on a Windows/MacOS computer, consider Mobaxterm/Cygwin (Windows) or XQuartz (MacOS).
